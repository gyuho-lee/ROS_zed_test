#### ROS_zed_test

This project purpose to capture zed stereo camera using OpenCV.

* Envrionment
  * Ubuntu 18.04
  * ROS melodic
  * OpenCV

* Source clone 
  * > git clone https://github.com/gyuho-lee/ROS_zed_test.git  

* Move to source directory
  * > cd ZED_opencv

* Source build
  * > catkin_make

* Run project node
  * > rosrun zed_opencv zed_opencv_node
